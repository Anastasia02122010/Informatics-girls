About us

Είμαστε μία ομάδα εφήβων κοριτσίων που γνωριστήκαμε μέσα απο το πρόγραμμα Code Like a Girl της Vodafone. Μας ένωσε η αγάπη για την τεχνολογία αλλά και η κοινή μας ανησυχία για το μέλλον του πλανήτη. Αποφασίσαμε να συνδυάσουμε τη δημιουργηκότητα και τις νέες δεξιότητες μας στον προγραμματισμό, για να φτιάξουμε αυτο το website με σκοπό να ενημερώνει, να εμπνέει και να κινητοποιεί. Θέλουμε να δείξουμε ότι ακόμα και η δική μας γενία μπορεί να συμβάλει σε μία πιο πράσινη και βιώσιμη καθημερινότητα.Στόχος μας είναι να προσφέρουμε πρακτικές ιδέες, δράσεις και πηγές γνώσης που θα βοηθήσουν όλους να κάνουν μικρά, αλλά σημαντικά βήματα για την προστασία του περιβάλλοντος.Γιατί το κάνουμε ; Γιατί πιστεύουμε ότι η αλλαγή ξεκινά απο εμάς και θέλουμε να γίνουμε η γεννία που θα αφήσει τον κόσμο καλύτερο απο ότι τον βρήκε.
